Introducción

Esta API proporciona acceso a información completa sobre el anime y manga Spy x Family, incluyendo temporadas, episodios, personajes y tomos.

Todas las respuestas están en formato JSON y siguen una estructura consistente.

Ejemplos de Uso

JavaScript (Fetch API)

fetch('{{ baseUrl }}/episodios')
  .then(response => response.json())
  .then(data => console.log(data))
  .catch(error => console.error(error));

cURL

curl -X GET {{ baseUrl }}/episodios

Angular HttpClient

this.http.get('{{ baseUrl }}/episodios')
  .subscribe({{'{'}}
    next: (data) => console.log(data),
    error: (error) => console.error(error)
  {{'}'}});
